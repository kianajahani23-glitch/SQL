# AI Impact on Students - SQL Analysis

## Project Overview
This project analyzes the impact of Generative AI usage on student academic performance using SQL queries executed with DuckDB in Google Colab.

## Dataset
## https://www.kaggle.com/datasets/laveshjadon/ai-impact-on-students/data

The dataset contains information about:
- Student GPA before and after the semester
- Weekly GenAI usage
- Prompt engineering skills
- AI dependency
- Study habits
- Burnout risk

## Tools Used
- SQL
- DuckDB
- Python
- Google Colab
- Pandas

## Analysis Performed
- GPA improvement analysis
- AI usage vs academic performance
- Prompt engineering skill analysis
- Burnout risk analysis
- Major category ranking
- Skill retention analysis

## Key Findings
- Students with stronger prompt engineering skills tended to achieve higher GPAs.
- AI dependency showed a relationship with skill retention.
- Academic outcomes varied across major categories and AI usage levels.
